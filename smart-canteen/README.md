# Smart Canteen Test Plan

This folder contains manual test cases for the Smart Canteen project.

## Modules covered
- Authentication
- Student flow
- Admin flow
- Orders
- Notifications
- Session and security
- API security
- UI responsiveness
- Deployment/production validation

## Status labels
- PASS
- FAIL
- BLOCKED
- NOT TESTED

## Suggested test format
For each test case, record:
- Test Case ID
- Module
- Preconditions
- Steps
- Expected Result
- Actual Result
- Status
- Notes
